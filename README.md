# mysql-booster

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)  
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)  

یک لایبرری ساده برای کار با MySQL در پایتون بدون نیاز به نوشتن SQL.

## فهرست مطالب

- ویژگی‌ها
- نصب
- استفاده
  - اتصال
  - SELECT
  - INSERT
  - UPDATE
  - DELETE
- شرط‌ها (where)
- مجوز

## ویژگی‌ها

- بدون نیاز به SQL
- پشتیبانی از SELECT, INSERT, UPDATE, DELETE
- شرط‌ها با suffix کاربردی
